# AstraNova eスポーツチーム公式サイト - 基本設計書

本ドキュメントは、AstraNova公式サイトにおける画面レイアウト、デザインシステム、カラー構成、タイポグラフィ、およびUIコンポーネントの基本設計を定義したものです。

---

## 1. サイト・ページ構成

シングルページ構成（1カラムロングスクロールレイアウト）を採用し、上部ナビゲーションから各セクションへスムーススクロールで遷移します。

### 1.1. セクション構成
1. **ローディング画面 (Loader)**: 初回アクセス時にブランドロゴとパーセンテージを描画。ロード後にフェードアウト。
2. **ナビゲーションヘッダー (Header)**: 固定配置。左側にチームロゴ、右側にアンカーナビゲーションおよびモバイル用ハンバーガーメニュー。
3. **ヒーローセクション (Hero)**: ファーストビュー。画面全体の背景ビジュアル、巨大キャッチコピー、CTAボタン、下部へのスクロール誘導。
4. **チーム紹介 (About Team)**: AstraNovaの概要、ミッション、主要活動内容を大きな文字と洗練された余白で表現。
5. **所属メンバー (Members)**: プレイヤー、ストリーマーのカード型ギャラリー。
6. **実績タイムライン (Achievements)**: 大会順位や主要実績をタイムライン形式で可視化。
7. **ギャラリー (Gallery)**: チーム活動、大会シーンなどのビジュアルグリッド。モーダル拡大表示機能付き。
8. **スポンサー (Sponsors)**: 協賛パートナーのロゴ一覧およびスポンサー募集セクション。
9. **お問い合わせ (Contact)**: 問い合わせフォームとCloudflare Turnstileウィジェット。
10. **フッター (Footer)**: コピーライト、各種SNSショートカット、法的免責事項へのリンク。

## 2. デザインシステム・スタイル設計

### 2.1. カラーパレット (CSSカスタムプロパティ)
高級感と近未来的なクールさを両立させるため、純粋な黒とダークグレーを基調とし、鮮やかだが絞り込まれたサイバーアクセントを採用します。

* **Base Background**: `#000000` (完全な黒 - 有機ELディスプレイ等でも映える深み)
* **Surface Background (Elevated)**: `#0a0a0a` (カードやセクション区分用の非常に暗いグレー)
* **Surface Border**: `#1a1a1a` (境界線用のシャープなダークグレー)
* **Primary Text**: `#ffffff` (純白)
* **Secondary Text**: `#8f8f8f` (可読性を損なわない中間グレー)
* **Accent Cyan**: `#00f3ff` (未来感を表す鮮やかなシアン。インタラクティブ要素の第一アクセント)
* **Accent Violet**: `#8f00ff` (神秘的かつ高級感を与えるバイオレット。グラデーションの第二アクセント)
* **Soft White Glow**: `rgba(255, 255, 255, 0.05)` (ホバー時などの柔らかい発光効果)

### 2.2. タイポグラフィ
Google Fontsから以下のWebフォントを読み込みます。

* **Heading Font**: `'Space Grotesk', sans-serif`
  - 特徴: 幾何学的でSF的な近未来感がありながら、極めて可読性が高くクリーン。
  - 用途: 大見出し、セクションタイトル、メンバー名など。
* **Body Font**: `'Plus Jakarta Sans', sans-serif`
  - 特徴: 現代的なプロポーションを持つ非常に美しいサンセリフ。AppleやNikeのような洗練された雰囲気を醸し出します。
  - 用途: 本文、説明文、フォーム入力など。

## 3. レスポンシブ設計ポリシー

モバイルファーストの視点を取り入れつつ、PCの大画面ディスプレイ（4K等）でも余白が崩れないように設計します。

* **ブレイクポイント**:
  - `Mobile`: `~767px`（1カラム表示、スワイプ操作対応、ハンバーガーメニュー表示）
  - `Tablet`: `768px ~ 1023px`（一部カードを2カラムに再配置、フォントサイズ調整）
  - `Desktop`: `1024px ~`（最大幅 `1200px` の中央揃えコンテナ、PC用横並びナビゲーション、ホバーエフェクトとカスタムカーソルの有効化）

## 4. UI/UX 演出設計

1. **カスタムカーソル (Custom Cursor)**:
   - PC環境でのみ動作。マウスカーソルに遅れて追従する直径30pxのリング（`mix-blend-mode: difference` を適用し、重なる要素の色を反転）を描画。
   - ボタンやリンクにホバーした際、リングが拡大し不透明度が変化するフィードバックを提供。
2. **スクロール進行度インジケーター**:
   - 画面最上部に配置された、現在のページスクロール位置をリアルタイムに示すシアン色の極細バー。
3. **スクロール出現アニメーション (Intersection Observer)**:
   - 対象要素に `.reveal` クラスを付与。画面に進入した時点で自動的に `.visible` が追加され、下から上への滑らかなスライドイン、またはフェードインが発火。
4. **画像モーダル (Lightbox)**:
   - ギャラリーの画像をクリックすると、背景がブラー（ぼかし）処理されたオーバーレイが開き、画像をフルスクリーン表示。キーボード（ESC）やモーダル外クリックでの閉鎖に対応するアクセシビリティ設計。
